Introduccion

Git es un sistema de control de versiones distribuido ampliamente utilizado.

Permite llevar un historial de cambios y colaborar con otros desarrolladores.

Fue creado por Linus Torvalds en 2005 para el desarrollo del kernel de Linux.

git --version git help
  • Sistema distribuido
  • Popular en desarrollo de software
  • Compatible con GitHub y GitLab
Instalacion

Git puede instalarse desde la web oficial o mediante gestores de paquetes.

Está disponible en Windows, macOS y Linux.

sudo apt install git brew install git
  • Descargar desde git-scm.com
  • Instalar vía terminal
Primeros Pasos

Una vez instalado, puedes configurar tu nombre y correo electrónico.

También puedes inicializar un nuevo repositorio con Git.

git config --global user.name "Tu Nombre" git config --global user.email "tu@email.com"
  • Configuración inicial
  • Inicializar repositorio
Comandos Comunes

Algunos comandos Git que debes conocer son:

Estos te permiten gestionar tus archivos y confirmar cambios.

git init git add . git commit -m "mensaje"
  • git status
  • git log
  • git push
Buenas Practicas

Es recomendable hacer commits pequeños y descriptivos.

Utiliza ramas para mantener organizado el desarrollo.

git checkout -b nueva-rama git merge rama
  • Commits frecuentes
  • Mensajes claros
  • Uso de ramas
